jenis-jenis Komputasi awan
  • Public cloud

      komputasi awan yang disediakan oleh penyedia layanan komputasi awan, seperti Amazon Web Services (AWS), Microsoft Azure, dan Google Cloud Platform. Public cloud menawarkan layanan komputasi awan yang terjangkau dan dapat diakses oleh siapa saja.

  • Private cloud

      komputasi awan yang dimiliki dan dioperasikan oleh organisasi tertentu. Private cloud menawarkan lebih banyak kontrol dan keamanan daripada public cloud, tetapi juga lebih mahal.

  • Hybrid cloud

      kombinasi dari public cloud dan private cloud. Hybrid cloud menawarkan fleksibilitas untuk menggunakan layanan public cloud dan private cloud sesuai kebutuhan.

Komputasi awan memungkinkan kita untuk mengakses dan menggunakan sumber daya komputasi dari mana saja, kapan saja. Hal ini dapat meningkatkan produktivitas dan efisiensi kerja, belajar, dan bermain. Misalnya, kita dapat menggunakan aplikasi komputasi awan untuk mengerjakan tugas-tugas kantor di rumah, atau mengikuti kelas online dari mana saja.

Komputasi awan dapat membantu meningkatkan aksesibilitas ke teknologi dan informasi. Misalnya, orang-orang di daerah terpencil dapat mengakses aplikasi dan layanan komputasi awan melalui internet. Hal ini dapat membantu mereka untuk tetap terhubung dengan dunia dan mengejar pendidikan dan peluang lainnya.

Komputasi awan dapat mendorong inovasi dengan menyediakan platform yang lebih terjangkau dan fleksibel untuk mengembangkan dan menyebarkan aplikasi dan layanan baru. Misalnya, startup dapat menggunakan komputasi awan untuk mengembangkan aplikasi dan layanan baru tanpa harus berinvestasi dalam infrastruktur komputasi yang mahal.

Komputasi awan dapat mengubah pola konsumsi kita. Misalnya, kita dapat berlangganan layanan komputasi awan berdasarkan penggunaan, sehingga kita hanya membayar untuk sumber daya yang kita gunakan. Hal ini dapat membantu kita menghemat uang dan mengurangi dampak lingkungan.
